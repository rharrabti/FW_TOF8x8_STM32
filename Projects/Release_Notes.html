Release Notes for Projects of X-CUBE-TOF1 Package

Copyright © 2022 STMicroelectronics

Purpose

The X-CUBE-TOF1 projects are a set of applicative examples that aim to explain and provide use cases of the VL53L1CB, VL53L3CX, VL53L4CX, VL53L4CD, VL53L5CX, VL53L7CX and VL53L8CX devices and nucleo boards. They provide basic functionalities, show main features or demonstrate performance. They are provided with preconfigured projects for the main supported toolchains. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals configuration.

Here is the list of references to the user documents:

  • X-NUCLEO-53L1A2: Multi-targets ranging sensor expansion board based on VL53L1 for STM32 Nucleo
  • VL53L1CB: Time-of-Flight ranging sensor with multi-object detection and FoV programming
  • VL53L1-SATEL: Breakout board for VL53L1 with multi-object detection and FoV programming
  • X-NUCLEO-53L3A2: Multi-targets ranging sensor expansion board based on VL53L3CX for STM32 Nucleo
  • VL53L3CX: Time-of-Flight ranging sensor with multi target detection
  • VL53L3CX-SATEL: Breakout board for VL53L3CX with multi-target detection
  • X-NUCLEO-53L4A2: Multi-targets ranging sensor expansion board based on VL53L4CX for STM32 Nucleo
  • VL53L4CX: Time-of-Flight ranging sensor with extended range measurement
  • SATEL-VL53L4CX: Breakout for VL53L4CX with extended range measurement
  • VL53L4CD: Time-of-Flight high-accuracy proximity sensor
  • SATEL-VL53L4CD: Breakout for VL53L4CD Time-of-Flight high-accuracy proximity sensor
  • X-NUCLEO-53L5A1: 8x8 multizone with wide field of view ranging sensor expansion board based on VL53L5CX for STM32 Nucleo
  • VL53L5CX: Time-of-Flight 8x8 multizone ranging sensor with wide field of view
  • VL53L5CX-SATEL: Breakout board for VL53L5CX
  • X-NUCLEO-53L7A1: 8x8 multizone with 90 degrees field of view ranging sensor expansion board based on VL53L7CX for STM32 Nucleo
  • VL53L7CX: Time-of-Flight 8x8 multizone ranging sensor with 90 degrees FoV
  • VL53L7CX-SATEL: Breakout board for VL53L7CX
  • X-NUCLEO-53L8A1: 8x8 multizone with 90 degrees field of view ranging sensor expansion board based on VL53L8CX for STM32 Nucleo
  • VL53L8CX: Time-of-Flight 8x8 multizone ranging sensor with 90 degrees FoV
  • VL53L8CX-SATEL: Breakout board for VL53L8CX

Known limitations

Headline
The 53L4A2_MultiSensorRanging, 53L3A2_MultiSensorRanging and 53L1A2_MultiSensorRanging sample applications don’t support multi target detection
When generating projects for STM32U5 it is possible that you will encounter compilation errors related to missing BUS_I2C1_SDA_GPIO_PORT and BUS_I2C1_SDA_GPIO_PIN symbols in custom_bus.h. When manually adding BUS_I2C1_SDA_GPIO_PORT and BUS_I2C1_SDA_GPIO_PIN the application compiles without errors.
