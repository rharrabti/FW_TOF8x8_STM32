Purpose

The VL53L1CB Component Driver is a driver intenteded to be used within the STM32Cube ecosystem. This software implements a Component Driver running on STM32. It is built on top of STM32Cube software technology that ease portability across different STM32 micro-controllers.

Here is the list of references to user documents:

  • VL53L1CB: Time-of-Flight ranging sensor with multi-object detection and FoV programming
  • VL53L1-SATEL: Breakout board with VL53L1 with multi-object detection and FoV programming
